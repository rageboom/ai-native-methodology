### C.2 PoC 대상 (다음 세션 확정)

권고: **RealWorld FE 구현체 중 React + TypeScript + (가능하면 TanStack Query + Zustand)**.

### C.3 7대 산출물 FE 특이 사항 (예상)

| 산출물           | BE                                        | FE (예상)                                                            |

| 02 도메인 모델   | Aggregate / Entity                        | View Model / Form Model / Server-state cache shape                   |

| 04 DB 스키마     | RDB schema                                | LocalStorage / IndexedDB / Zustand store schema                      |
| 04.5 형식명세    | state-machine + sequence + decision table | UI state machine (form / route / async loading)                      |
| 05 비즈니스 규칙 | rules.json                                | Form validation rules / Permission rules / UI display rules          |
